Typedef Documentation

typedef void(* noise::utils::NoiseMapCallback)(int row)

A pointer to a callback function used by the NoiseMapBuilder class.

The NoiseMapBuilder::Build() method calls this callback function each time it fills a row of the noise map with coherent-noise values.

This callback function has a single integer parameter that contains a count of the rows that have been completed. It returns void. Pass a function with this signature to the NoiseMapBuilder::SetCallback() method.

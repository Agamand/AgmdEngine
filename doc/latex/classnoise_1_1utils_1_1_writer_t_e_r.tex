\subsection{Detailed Description}
Terragen Terrain writer class.

This class creates a file in Terrage Terrain ($\ast$.ter) format given the contents of a noise map object. This class treats the values in the noise map as elevations measured in meters.

\href{http://www.planetside.co.uk/terragen/}{\tt Terragen} is a terrain application that renders realistic landscapes. Terragen is available for Windows and Mac\+O\+S; unfortunately, Terragen does not have U\+N$\ast$\+X versions.

{\bfseries Writing the noise map}

To write the noise map, perform the following steps\+:
\begin{DoxyItemize}
\item Pass the filename to the \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_af8db9e0a480658949ba8d37f4cb03368}{Set\+Dest\+Filename()} method.
\item Pass a \hyperlink{classnoise_1_1utils_1_1_noise_map}{Noise\+Map} object to the \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_a26314e0c188948fa01200a54189c28ca}{Set\+Source\+Noise\+Map()} method.
\item Call the \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_ad4332c36a48cb32bef6798def109052e}{Write\+Dest\+File()} method.
\end{DoxyItemize}

The \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_af8db9e0a480658949ba8d37f4cb03368}{Set\+Dest\+Filename()} and \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_a26314e0c188948fa01200a54189c28ca}{Set\+Source\+Noise\+Map()} methods must be called before calling the \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_ad4332c36a48cb32bef6798def109052e}{Write\+Dest\+File()} method.

This method encodes the contents of the noise map and writes it to a file. Before calling this method, call the \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_a26314e0c188948fa01200a54189c28ca}{Set\+Source\+Noise\+Map()} method to specify the noise map, then call the \hyperlink{classnoise_1_1utils_1_1_writer_t_e_r_af8db9e0a480658949ba8d37f4cb03368}{Set\+Dest\+Filename()} method to specify the name of the file to write.

This object assumes that the noise values represent elevations in meters.

A pointer to a callback function used by the \hyperlink{classnoise_1_1utils_1_1_noise_map_builder}{Noise\+Map\+Builder} class.

The \hyperlink{classnoise_1_1utils_1_1_noise_map_builder_abe0e5f01c696c2290792c8f8736e3ed1}{Noise\+Map\+Builder\+::\+Build()} method calls this callback function each time it fills a row of the noise map with coherent-\/noise values.

This callback function has a single integer parameter that contains a count of the rows that have been completed. It returns void. Pass a function with this signature to the \hyperlink{classnoise_1_1utils_1_1_noise_map_builder_ac760187423eb5e16ac015f9beafd78ea}{Noise\+Map\+Builder\+::\+Set\+Callback()} method.
